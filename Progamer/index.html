Você quer se desenvolvedor web front-end? Sabe por onde começar?

qual são os requisitos para se torna desenvolvedor web front-end?

  1. Um pc, que contenha o windows 7, ou 8, ou 10, ou 11, ou linux, etc. Caso você não tenha nenhum desses sistemas, você tera que progamar da forma antiga, que é usando o famoso "Bloco de Notas", mas não se preocupe, porque os códigos serão os mesmos.

  2. Ter acesso a um wi-fi. para progamar em html css e javascript, não é necessario ter um wi-fi, mas ele é bem útil para realizar pesquisas, e o mais importante, para fazer a hospedagem do site.

  3. Seu pc tem que ter pelo menos 500mb para instalar algumas ferramentas, como Google chrome (navegador mais utilizado no brasil) e o Visual Studio Code (Editor de código)
